# Antigravity Coordinator

Self-optimizing multi-agent coordinator with DQ scoring, ACE consensus, and research-driven feedback loops.

## Install

```bash
pip install antigravity-coordinator
```

## Quick Start

```bash
coord init                    # Initialize ~/.coordinator/
coord research "task"         # 3 parallel explore agents
coord implement "task"        # Parallel builders with file locks
coord auto "task"             # Auto-detect strategy
coord status                  # Show active agents
coord history                 # Session outcomes with DQ scores
coord optimize --dry-run      # Propose self-improvements
```
